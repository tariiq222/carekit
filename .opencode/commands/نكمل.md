# /نكمل — Resume Paused Task

## What This Command Does

Resumes ONLY the last task that was paused via `/نوقف`.
This command is only valid when the user explicitly types `/نكمل`.
It must never be triggered automatically or inferred from context.

---

## Trigger

User types: `/نكمل`

This is the ONLY trigger. No other input activates resume behavior.

---

## Behavior — Decision Tree

### Step 1: Read the state file

Read: `.opencode/context/current-task.json`

### Step 2: Check `status` field

**If file does not exist or `status` is `"none"`:**
```
لا يوجد مهمة موقوفة للاستئناف.
ابدأ مهمة جديدة بشكل طبيعي.
```
Stop. Do not start a new task.

**If `status` is `"done"`:**
```
المهمة الأخيرة ([task]) مكتملة بالفعل.
لا يمكن إعادة فتحها.
ابدأ مهمة جديدة بشكل طبيعي إذا أردت.
```
Stop. Do not reopen it.

**If `status` is `"paused"`:**
Proceed to Step 3.

### Step 3: Confirm to user before resuming

Display:
```
⏩ استئناف المهمة:

المهمة   : [task]
المرحلة  : [current_stage]
التقدم   : [progress]
الخطوة التالية: [next_steps[0]]

هل تريد المتابعة؟ (نعم / لا)
```

Wait for confirmation. Do not proceed until user confirms.

### Step 4: Resume from `current_stage`

Route through the CTO orchestration pipeline starting at `current_stage`.

- If `current_stage` is `analyze` → run analyze-task pipeline
- If `current_stage` is `implement` → run implement-plan pipeline (plan already exists)
- If `current_stage` is `test` → run write-tests pipeline
- If `current_stage` is `review` → run review-diff pipeline
- If `current_stage` is `qa` → run qa-check pipeline

Load context from the saved fields:
- `decisions` → feed into architect/executor context
- `files` → the files already modified
- `tests_added` → tests already written
- `next_steps` → the specific next action
- `blockers` → surface to user before executing if non-empty

### Step 5: After successful completion


If QA passes and memory is written, update the file:
```json
{
  "status": "done"
}
```

---

## Hard Rules

- `/نكمل` resumes ONLY — it never starts a new independent task
- `/نكمل` must not create a plan, analyze a new problem, or generate code for anything other than the saved task
- If `blockers[]` is non-empty, surface them to the user BEFORE executing — do not proceed past them silently
- If `current_stage` was `implement` and `files[]` is non-empty, read those files first before executing the next step
- Model routing is automatic — each agent uses its own model. No manual switch required.

---

## What /نكمل Is NOT

- It is NOT a generic workflow starter
- It is NOT triggered by any other phrase or command
- It is NOT auto-activated when a session opens
- It is NOT activated if the user describes the task again without typing `/نكمل`

---

## Example — Task exists and is paused

User opens new session, types: `/نكمل`

System reads `.opencode/context/current-task.json`:
```json
{
  "task": "Fix no-show block not enforced on walk-in bookings",
  "status": "paused",
  "current_stage": "implement",
  "progress": "Architect analysis complete. Executor has modified BookingsService.create().",
  "decisions": ["Walk-in bookings must also check no-show status"],
  "files": ["backend/src/modules/bookings/bookings.service.ts"],
  "tests_added": [],
  "next_steps": [
    "Write regression test in backend/src/modules/bookings/tests/bookings.service.spec.ts"
  ],
  "blockers": [],
  "runtime": {
    "confidence_score": 92,
    "escalation_recommended": false,
    "escalation_reason": "",
    "token_risk_level": "low",
    "context_file_count": 3
  }
}
```

System displays:
```
⏩ استئناف المهمة:

المهمة   : Fix no-show block not enforced on walk-in bookings
المرحلة  : implement
التقدم   : Architect analysis complete. Executor has modified BookingsService.create().
الخطوة التالية: Write regression test in backend/src/modules/bookings/tests/bookings.service.spec.ts

هل تريد المتابعة؟ (نعم / لا)
```

User confirms → resumes at `implement` stage with loaded context.

---

## Example — No paused task

User opens new session, types: `/نكمل`

File has `status: "none"` or doesn't exist.

System says:
```
لا يوجد مهمة موقوفة للاستئناف.
ابدأ مهمة جديدة بشكل طبيعي.
```

Stops. Does not start anything.
