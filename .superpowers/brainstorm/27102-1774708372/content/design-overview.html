المحور 1 — color-utils.ts: اشتقاق Dark Variants

إضافة HSL conversion + darkVariant()

hexToHsl(hex) → { h, s, l }
hslToHex({ h, s, l }) → hex

darkVariant(color, targetL = 0.68):
  → نفس الـ hue + saturation
  → lightness يُرفع لـ 68% (مقروء على dark bg)
  → إذا اللون أصلاً فاتح كفاية → يُخفَّف saturation بدل رفع lightness

deriveCssVars() تُرجع الآن light vars + dark vars في نفس الـ object

المحور 2 — branding-provider.tsx: حقن Dark Vars

injectVars() تنقسم لوظيفتين:

injectLightVars(vars) → تكتب على document.documentElement (كما هو)
injectDarkVars(vars) → تكتب <style> tag بـ .dark { ... }
                       id="carekit-dark-theme" (لإعادة الاستخدام)

clearVars() تحذف الـ style tag أيضاً — لا تراكم
