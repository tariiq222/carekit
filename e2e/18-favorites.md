# المفضلة (Favorites)

| # | الاسم | الوصف | النتيجة المتوقعة |
|---|-------|-------|-----------------|
| F1 | إضافة للمفضلة | مريض يضيف طبيباً | 201 + `employeeId` |
| F2 | قراءة المفضلة | GET /favorites | 200 + `length >= 1` |
| F3 | شكل الاستجابة | كل طبيب يحتوي `id + specialty + user` | حقول موجودة |
| F4 | بدون توكن | إضافة بدون auth | 401 AUTH_TOKEN_INVALID |
| F5 | إضافة مكررة | نفس الطبيب مرتين | 409 CONFLICT |
| F6 | إزالة من المفضلة | DELETE /favorite | 200 + `success: true` |
| F7 | لا يظهر بعد الإزالة | قراءة المفضلة بعد الحذف | الطبيب غائب |
| F8 | إزالة غير موجود | حذف بعد إزالة مسبقة | 404 NOT_FOUND |
| F9 | قراءة بدون توكن | GET /favorites بدون auth | 401 AUTH_TOKEN_INVALID |
| F10 | عزل بين المرضى | كل مريض يرى مفضلته فقط | مصفوفتان مستقلتان |
| F11 | طبيب وهمي | إضافة UUID وهمي | 404 EMPLOYEE_NOT_FOUND |
