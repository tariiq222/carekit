# التخصصات (Specialties)

## إنشاء تخصص

| # | الاسم | الوصف | النتيجة المتوقعة |
|---|-------|-------|-----------------|
| SP-C1 | إنشاء أساسي | nameEn + nameAr | 201 + specialtyId |
| SP-C2 | مع وصف | descriptionEn + descriptionAr | 201 + محفوظ |
| SP-C3 | مع أيقونة | `iconUrl` | 201 + محفوظ |
| SP-C4 | مع ترتيب | `sortOrder: 5` | 201 + الترتيب محفوظ |
| SP-C5 | بدون nameEn | حقل إلزامي | 400 VALIDATION_ERROR |
| SP-C6 | بدون nameAr | حقل إلزامي | 400 VALIDATION_ERROR |
| SP-C7 | اسم مكرر | name موجود مسبقاً | 409 CONFLICT |
| SP-C8 | sortOrder سالب | `sortOrder: -1` | 400 VALIDATION_ERROR |

---

## قراءة التخصصات (عام — بدون auth)

| # | الاسم | الوصف | النتيجة المتوقعة |
|---|-------|-------|-----------------|
| SP-L1 | قراءة الكل | GET /specialties | 200 + مصفوفة |
| SP-L2 | تخصص بـ ID | GET /specialties/:id | 200 + التفاصيل |
| SP-L3 | ID وهمي | UUID غير موجود | 404 NOT_FOUND |
| SP-L4 | UUID تنسيق خاطئ | `id: "abc"` | 400 VALIDATION_ERROR |

---

## تعديل تخصص

| # | الاسم | الوصف | النتيجة المتوقعة |
|---|-------|-------|-----------------|
| SP-U1 | تعديل الاسم | تغيير nameEn | 200 + الاسم الجديد |
| SP-U2 | تعطيل التخصص | `isActive: false` | 200 + لا يظهر للعموم |
| SP-U3 | تعديل الترتيب | تغيير sortOrder | 200 + الترتيب محدّث |
| SP-U4 | اسم مكرر | name موجود عند تخصص آخر | 409 CONFLICT |
| SP-U5 | تخصص غير موجود | ID وهمي | 404 NOT_FOUND |

---

## حذف تخصص

| # | الاسم | الوصف | النتيجة المتوقعة |
